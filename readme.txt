== Description ==

WP Alex is a plugin which will integrate [Alex](http://alexjs.com) into WordPress's TinyMCE editor. Alex alerts you to insensitive and inconsiderate words and phrases within your post.

== Frequently Asked Questions ==

= Is there a way to minimize the notifcations? =

Simply click on Alex in the editor and it should toggle the visibility of the notifications.

= Does this support multiple editors? =

At the moment, this just supports the default WP editor. Additional editors added to the page through plugins or custom code may interfere with WP Alex's functionality.

= Why does this work with the text view of the editor? =

This is feature planned for future releases.
